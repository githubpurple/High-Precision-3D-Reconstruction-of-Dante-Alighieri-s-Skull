# High-Precision-3D-Reconstruction-of-Dante-Alighieri-s-Skull
## Description
- Performed high-fidelity 3D modeling and reverse engineering of an anthropological relic (Dante’s skull) using Geomagic Design X and mesh processing pipelines.
- Optimized complex geometric meshes for morphological analysis, ensuring sub-millimeter accuracy required for bio-archaeological research.
- Demonstrated advanced proficiency in managing the full digital workflow: from raw scan data acquisition to creating research-grade digital twins suitable for virtual reality (VR) and digital archiving.
